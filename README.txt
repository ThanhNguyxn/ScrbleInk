## Hướng dẫn cài đặt (Tiếng Việt)

1. Tải xuống các tệp sau:
   - Scrible_thanhnguyxn.pfx (Chứng chỉ bảo mật)
   - Scrible_thanhnguyxn.cer (Chứng chỉ công khai)
   - Scrible thanhnguyxn.appx (File cài đặt ứng dụng)

2. Cài đặt chứng chỉ:
   - Nhấp đúp vào file Scrible_thanhnguyxn.cer
   - Chọn "Install Certificate"
   - Chọn "Local Machine"
   - Chọn "Place all certificates in the following store"
   - Chọn "Trusted Root Certification Authorities"
   - Nhấn "Next" và "Finish"

3. Cài đặt ứng dụng:
   - Nhấp đúp vào file "Scrible thanhnguyxn.appx"
   - Làm theo các bước trong trình cài đặt

## Installation Guide (English)

1. Download the following files:
   - Scrible_thanhnguyxn.pfx (Security certificate)
   - Scrible_thanhnguyxn.cer (Public certificate)
   - Scrible thanhnguyxn.appx (App installer)

2. Install the certificate:
   - Double-click on Scrible_thanhnguyxn.cer
   - Select "Install Certificate"
   - Choose "Local Machine"
   - Select "Place all certificates in the following store"
   - Choose "Trusted Root Certification Authorities"
   - Click "Next" and "Finish"

3. Install the app:
   - Double-click on "Scrible thanhnguyxn.appx"
   - Follow the installation steps
